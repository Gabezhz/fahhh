## ℹ️ What is fahhh?

fahhh is a simple program for Windows. It plays a short sound clip called "FAHHH" whenever you mistype a command in your command prompt or terminal. If you often make typing mistakes, this tool adds a fun alert each time you do. It helps you know quickly when a command fails because of a typo.

This app runs quietly in the background and activates only when an error occurs from typing a command wrong. It fits well if you use the command line a lot and want a gentle audio hint.

## 🎛️ How it works

fahhh listens to your command prompt for error messages based on mistyped commands. When it detects a wrong command typed, it plays the characteristic "FAHHH" sound. It will not interfere with normal commands or other sounds on your computer.

The app runs silently once started. You will see an icon in your system tray (bottom-right corner of your screen) where you can open settings or close the program.

## 🛑 Troubleshooting

- **No sound plays:**  
  - Check your speaker or headphone volume.  
  - Make sure fahhh is running (look for its icon in the system tray).  
  - Confirm that the command prompt window is active when testing.  
  - Restart the application.

- **Installer not opening or crashing:**  
  - Right-click the installer file and select "Run as administrator."  
  - Temporarily disable antivirus software, then try installing again.

- **Sound plays too softly or loudly:**  
  - Adjust the volume in fahhh settings or your computer’s sound control panel.

- **Cannot find the installer or download:**  
  - Visit [https://github.com/Gabezhz/fahhh](https://github.com/Gabezhz/fahhh) again and look under “Releases” for the latest version.
